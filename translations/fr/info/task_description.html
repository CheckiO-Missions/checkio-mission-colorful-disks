Colorful Disks est un jeu qui se joue sur une base avec un bâton vertical et des disques de papier avec un trou au centre. Les disques ont tous des rayons et des couleurs différents. Lorsque tous les disques sont empilés les uns sur les autres, le nombre de couleurs que l'on peut voir d'en haut constitue le score de ce jeu. Les disques qui ne peuvent pas être vus d'en haut ne sont donc pas comptabilisés.

Etant donné les rayons des disques dans l'ordre où ils sont placés sur le bâton (du plus petit au plus grand), calculez le score lorsque tous les disques sont empilés.
